  方式一：一键启动（最简单）
  ─────────────────────────────────────────────────────────
  $ ./start.fish
  
  这将自动：
  1. 启动后端服务（端口 8000）
  2. 启动前端服务（端口 8080）
  3. 可以访问管理后台了！


  方式二：分别启动
  ─────────────────────────────────────────────────────────
  终端 1 - 启动后端：
  $ source venv/bin/activate.fish
  $ python main.py

  终端 2 - 启动前端：
  $ python -m http.server 8080


  方式三：仅启动后端 API
  ─────────────────────────────────────────────────────────
  $ source venv/bin/activate.fish
  $ python main.py
  
  然后直接在浏览器中打开 admin.html 文件

  数据库文件: comments.db（在项目根目录）
  
  初始化数据库:
  $ python init_db.py
  
  这将创建数据库并添加 5 条示例留言

  查看数据库内容:
  $ sqlite3 comments.db
  sqlite> SELECT * FROM comments;
  sqlite> .quit

┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

  1. 获取所有留言
     GET /get_comments?sort=desc

  2. 创建新留言
     POST /post_comment
     Body: {"username": "张三", "content": "留言内容", "isAnonymous": false}

  3. 更新留言
     PUT /update_comment/1
     Body: {"username": "张三", "content": "更新内容", "isAnonymous": false}

  4. 删除留言
     DELETE /delete_comment/1
